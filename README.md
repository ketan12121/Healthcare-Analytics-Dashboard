# Healthcare Analytics Dashboard

## Project Overview

This Healthcare Analytics Dashboard was developed using Power BI to analyze hospital performance, patient demographics, treatment costs, insurance coverage, and readmission trends. The dashboard provides healthcare stakeholders with actionable insights to improve operational efficiency and support data-driven decision-making.

## Tools & Technologies

* Power BI
* SQL Server
* DAX
* Excel
* Data Cleaning & Transformation

## Dataset Information

The dataset contains 10,000 patient records, including:

* Patient Demographics
* Disease Information
* Department Details
* Treatment Costs
* Insurance Status
* Readmission Status
* Patient Outcomes
* Monthly Admissions

## Key Performance Indicators (KPIs)

* Total Patients
* Total Revenue
* Average Treatment Cost
* Average Length of Stay
* Average Patient Satisfaction Score

## Dashboard Features

### Executive Dashboard

* Department-wise Patient Analysis
* Department-wise Revenue Analysis
* Disease Distribution Analysis
* Monthly Patient Admissions Trend
* Insurance Coverage Breakdown
* Interactive Filters and Slicers

### Patient Insights Dashboard

* Age Group Analysis
* Gender Distribution
* Average Treatment Cost by Disease
* Readmission Analysis
* Patient Outcome Analysis

## Key Insights

* Neurology and General Medicine recorded the highest patient volumes.
* Pediatrics generated the highest revenue among departments.
* Insurance coverage was almost evenly distributed between insured and uninsured patients.
* Heart Disease and Stroke showed higher readmission counts compared to other diseases.
* The majority of patients belonged to the 65+ age group.

## Business Impact

The dashboard enables healthcare administrators to:

* Monitor hospital performance in real time
* Track patient admission trends
* Evaluate departmental revenue contribution
* Analyze treatment costs across diseases
* Improve resource allocation and operational planning
